## ♻️ How to contribute

```bash
    # Clone the project
    $ git clone https://github.com/Luan-Costa-2l/whatsappclone
```

```bash
    # Open the directory
    $ cd whatsappclone
```

```bash
    # Install the dependecies
    $ npm install
```

```bash
    # Run the project
    $ npm run dev
```
